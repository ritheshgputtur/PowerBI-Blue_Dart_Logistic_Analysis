# BlueDart Logistics Analysis (Power BI)

## Overview
This project analyzes logistics and delivery performance using Power BI. It focuses on key metrics such as delivery efficiency, delays, customer satisfaction, and operational performance.

## Key Metrics
- Total Orders: 13,976  
- On-Time Delivery: 79.9%  
- Customer Satisfaction (CSAT): 84.7%  
- Average Delivery Time: 35.59 hours  

## Features
- Interactive dashboard with filters (Year, Month, Vehicle Type)
- Analysis of delivery performance and delays
- Hub-wise and vehicle-wise performance insights
- Driver performance and delay tracking

## Project Structure
- `data/` – Raw dataset used for analysis  
- `images/` – images used in report  
- `bluedart-logistics-dashboard.pbix` – Power BI file  
- `bluedart-report.pdf` – Exported report  

## Key Insights
- High delivery success rate, but on-time performance needs improvement  
- Vehicle breakdown and road conditions are major delay factors  
- Longer delivery times reduce customer satisfaction  
- Older vehicles and certain drivers contribute more to delays  

## Tools Used
- Power BI  
- Data Cleaning & Transformation (Power Query)  
- Data Modeling & Visualization  

## Objective
To analyze logistics operations and identify key factors affecting delivery performance and customer satisfaction.
